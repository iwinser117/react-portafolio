# Portafolio Web - Iwinser Sanchez

Portafolio personal desarrollado con React para presentar perfil profesional, proyectos, artículos técnicos y canales de contacto en una sola experiencia web.

## Sitio en producción

<https://iwinsersanchez.netlify.app/>

## ¿Qué es este proyecto?

Este proyecto es una plataforma de marca personal enfocada en mostrar:

- Perfil profesional y propuesta de valor.
- Habilidades técnicas y experiencia.
- Proyectos destacados con enlaces a demo y repositorio.
- Formulario de contacto integrado para facilitar comunicación directa.

## ¿Qué hace?

La aplicación funciona como un portafolio dinámico con navegación por rutas y componentes reutilizables. Sus funcionalidades principales son:

- Página principal con secciones informativas (presentación, acerca de, habilidades, contacto y footer).
- Vista de aplicaciones/proyectos con:
  - Búsqueda por texto.
  - Filtro por categorías.
  - Tarjetas con imagen, etiquetas, enlace a demo y enlace a repositorio.
  - Modo visual adaptado al tema activo.
- Blog con listado moderno de artículos y vista por slug.
- Sistema de internacionalización para contenido multilenguaje.
- Soporte de tema oscuro/claro mediante proveedor global.
- Diseño responsive para diferentes tamaños de pantalla.

## Estructura funcional

- Home: entrada principal del portafolio y secciones de presentación.
- Aplicaciones: galería de proyectos con filtros y buscador.
- Blog: listado de artículos y detalle de cada publicación.

## Tecnologías principales

- React 18
- React Router DOM
- i18next + react-i18next
- Webpack 5
- CSS/SASS
- MUI (Material UI)
- EmailJS

## Objetivo

Comunicar de forma clara el perfil técnico de Iwinser Sanchez, centralizando proyectos, contenido técnico y puntos de contacto en una sola web pública.
